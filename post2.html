Exploring Logistic Regression: A Case Study in Medical Diagnosis

Published on April 17, 2025 by Kalani Hasanthika

Abstract

This article demonstrates logistic regression through a case study in medical diagnosis. We use the Pima Indians Diabetes dataset to model the probability of a diabetes diagnosis based on several health parameters. The methodology covers data preprocessing, model fitting, and evaluation using R. Key findings illustrate the effectiveness of logistic regression for binary classification tasks in a medical context, and we discuss the strengths, limitations, and potential implications of our results.

Introduction

Medical diagnosis often involves predicting disease presence based on patient characteristics. Logistic regression is a widely used method in such scenarios, due to its interpretability and efficiency for binary outcomes. In this article, we focus on logistic regression to predict diabetes using the Pima Indians Diabetes dataset, a freely available dataset that contains diagnostic measurements such as blood pressure, body mass index, and glucose concentration. We discuss the theory behind logistic regression, provide a detailed walk-through of the methodology using R, and interpret the results in the context of medical diagnosis.

Background / Theory

Logistic regression estimates the probability that a binary response variable \(Y\) equals 1 given a set of predictor variables \(X\). Unlike linear regression, logistic regression uses the logistic (sigmoid) function to restrict predictions between 0 and 1:

\[ P(Y=1 \mid X) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 X_1 + \cdots + \beta_k X_k)}} \]

Key points include:

  • Interpretability: Model coefficients can be translated into odds ratios.
  • Assumptions: Independence among predictors, a linear relationship between the log odds and predictors, and minimal multicollinearity.
  • Application: Particularly useful for binary outcomes such as disease diagnosis (e.g., diabetic vs. non-diabetic).

Dataset Description

The Pima Indians Diabetes dataset provides a practical example for binary classification. It contains 768 observations and 9 variables, with the following features:

  • Pregnancies: Number of times pregnant
  • Glucose: Plasma glucose concentration
  • BloodPressure: Diastolic blood pressure (mm Hg)
  • SkinThickness: Triceps skin fold thickness (mm)
  • Insulin: 2-Hour serum insulin (mu U/ml)
  • BMI: Body mass index (kg/m²)
  • DiabetesPedigreeFunction: Diabetes pedigree function
  • Age: Age (years)
  • Diabetes: Class variable (“pos” for diabetic and “neg” for non-diabetic)

This dataset is originally hosted in various public repositories and is available in R via the mlbench package.

Methodology

We now demonstrate the analysis in R. The following R code shows how to load the dataset, partition the data into training and test sets, fit the logistic regression model, and evaluate its performance using both model summary statistics and an ROC curve.

# Partition the dataset into training (75%) and testing (25%) sets
set.seed(123)
train_index <- createDataPartition(dataset$diabetes, p = 0.75, list = FALSE)
train_data <- dataset[train_index, ]
test_data  <- dataset[-train_index, ]

# Fit a logistic regression model using all predictors
model <- glm(diabetes ~ ., data = train_data, family = binomial)

# Display a summary of the model
summary(model)

# Make predictions on the test set: obtain probabilities
probabilities <- predict(model, newdata = test_data, type = "response")

# Convert probabilities into binary predictions (threshold = 0.5)
predicted_classes <- ifelse(probabilities > 0.5, "pos", "neg")
confusion_matrix <- table(Predicted = predicted_classes, Actual = test_data$diabetes)
print(confusion_matrix)
##          Actual
## Predicted neg pos
##       neg 108  26
##       pos  17  41
# Evaluate model performance using ROC curve
pred <- prediction(probabilities, test_data$diabetes)
perf <- performance(pred, "tpr", "fpr")

# Plot the ROC curve
plot(perf, main = "ROC Curve for Logistic Regression Model", col = "blue", lwd = 2)
abline(a = 0, b = 1, col = "gray", lty = 2)  # Diagonal line for random classifier

# Calculate the Area Under the Curve (AUC)
auc <- performance(pred, measure = "auc")
auc_value <- auc@y.values[[1]]
print(paste("AUC:", round(auc_value, 3)))
## [1] "AUC: 0.845"

Explanation of the Code

  • Data Partitioning: We split the data into training and testing subsets to validate our model on unseen data.
  • Model Fitting: The glm function fits a logistic regression model using the binomial family.
  • Model Evaluation: We assess model performance through a confusion matrix and an ROC curve. The ROC curve and the computed AUC value provide insight into the model’s discrimination ability.

Results

The output of the logistic regression model is summarized by the model coefficients and their significance levels. Key performance indicators from the testing phase include:

  • Confusion Matrix: Provides counts of true positives, true negatives, false positives, and false negatives.
  • ROC Curve & AUC: The ROC curve visually represents the trade-off between sensitivity and specificity. An AUC value closer to 1 indicates excellent model performance. In our case, the AUC (printed in the R console) reflects the model’s diagnostic ability.

Discussion

The logistic regression model effectively utilizes the diagnostic features from the dataset to predict diabetes. Notable points include:

  • Interpretation of Coefficients: Significant predictors—such as plasma glucose concentration—demonstrated strong associations with diabetes risk.
  • Model Performance: An AUC in the mid- to high-70s (or above) suggests that the model performs well in distinguishing between diabetic and non-diabetic cases.
  • Limitations: While logistic regression provides robust estimates for binary classification, limitations such as potential multicollinearity among predictors or nonlinear relationships may affect the results.
  • Practical Implications: This approach can assist clinicians in evaluating diagnostic measures and refining screening protocols, though further validation is essential for real-world clinical application.

Conclusion

This case study showcased the application of logistic regression to a real-world medical diagnosis dataset. We highlighted the steps from data exploration and preprocessing through model fitting and evaluation using R. Although the Pima Indians Diabetes dataset provides an accessible platform for such analysis, further studies with larger, more diverse datasets would help validate and extend these findings. Logistic regression remains a valuable tool in medical statistics, offering both interpretability and practical relevance.
